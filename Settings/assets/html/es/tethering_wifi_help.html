Anclaje a red WLAN

Puedes convertir el teléfono en un punto de acceso WLAN para compartir la conexión a Internet del teléfono con uno o con varios equipos o dispositivos.
  • Cuando el teléfono actúe como un punto de acceso, no podrás utilizar sus aplicaciones para acceder a Internet a través de la conexión WLAN del dispositivo.

  • Para configurar un punto de acceso WLAN, accede a la configuración de anclaje a red WLAN.

  • Para obtener más información, consulta la página http://www.android.com/tether#wifi.
