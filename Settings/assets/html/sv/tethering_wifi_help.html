Internetdelning via WLAN

Du kan göra din telefon till en åtkomstpunkt för WLAN om du vill dela telefonens Internetanslutning med en eller flera datorer eller enheter
  • När telefonen fungerar som åtkomstpunkt kan du inte använda telefonens program med Internet via telefonens WLAN-anslutning

  • Du konfigurerar en WLAN-åtkomstpunkt under inställningarna för Internetdelning via WLAN

  • Om du vill veta mer besöker du http://www.android.com/tether#wifi
